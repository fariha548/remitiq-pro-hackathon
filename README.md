# 🏦 RemitIQ Pro
### Multi-Agent AI Remittance Operating System

> Google Gen AI Academy APAC — Cohort 1 Hackathon

## 🎯 Problem
Pakistan receives $30 Billion+ in annual remittances.
Bank officers manage 200+ tasks manually every week.

## 💡 Solution
RemitIQ Pro is a multi-agent AI system that automates
the entire remittance bank workflow automatically.

## 🤖 Agents
- Primary Coordinator — Orchestrates all agents
- Task Agent — MCP: Google Tasks API
- Calendar Agent — MCP: Calendar Tool
- Notes Agent — MCP: Notes Tool
- Compliance Agent — MCP: Compliance Tool

## 🛠️ Tech Stack
- Google ADK — Agent framework
- Gemini 3 Flash Preview — AI model
- Cloud Firestore — Database
- Cloud Run — Deployment
- Google Tasks API — Real MCP
- FastAPI — Backend
- Tailwind CSS — Frontend
